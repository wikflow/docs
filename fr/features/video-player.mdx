Le lecteur video Wikio prend en charge les raccourcis clavier habituels et des controles precis au niveau de l'image.

## Raccourcis de lecture

| Touche | Action |
|---|---|
| <kbd>Espace</kbd> / <kbd>K</kbd> | Lecture / Pause |
| <kbd>J</kbd> | Reculer de 10 s |
| <kbd>L</kbd> | Avancer de 10 s |
| <kbd>←</kbd> | Reculer de 5 s |
| <kbd>→</kbd> | Avancer de 5 s |
| <kbd>Shift</kbd>+<kbd>←</kbd> | Image precedente (en pause) |
| <kbd>Shift</kbd>+<kbd>→</kbd> | Image suivante (en pause) |
| <kbd>,</kbd> | Image precedente (en pause) |
| <kbd>.</kbd> | Image suivante (en pause) |

## Volume et affichage

| Touche | Action |
|---|---|
| <kbd>M</kbd> | Couper / Retablir le son |
| <kbd>↑</kbd> | Augmenter le volume |
| <kbd>↓</kbd> | Diminuer le volume |
| <kbd>F</kbd> | Basculer en plein ecran |
| <kbd>C</kbd> | Afficher / Masquer les sous-titres |

## Vitesse

| Touche | Action |
|---|---|
| <kbd>Shift</kbd>+<kbd>></kbd> | Augmenter la vitesse |
| <kbd>Shift</kbd>+<kbd><</kbd> | Diminuer la vitesse |

## Format temporel

Basculez entre le format **standard** (hh:mm:ss), **timecode** (hh:mm:ss:ff) et **images** depuis le menu des parametres du lecteur.

## Outils d'edition

Lorsque vous travaillez dans l'editeur de transcription ou de sous-titres, des outils supplementaires sont disponibles dans la barre d'outils du lecteur :

- **Lecture en boucle** -- repete le segment actuellement selectionne.
- **Definir un point d'entree / sortie** -- marquez une portion de la timeline pour l'export ou la relecture.
- **Vue forme d'onde** -- activez un affichage de la forme d'onde audio sous la video pour un montage precis.
